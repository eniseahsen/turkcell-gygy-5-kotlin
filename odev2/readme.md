# Kotlin OOP Project


Bu proje, Kotlin dilinde Nesne Yönelimli Programlama (OOP) kavramlarını öğrenmek ve uygulamak amacıyla hazırlanmıştır.

---

## Kullanılan OOP Kavramları

### 1.[Encapsulation (Kapsülleme)](src/encapsulation) 
Verilerin dış erişime kapatılması ve kontrollü şekilde yönetilmesidir.

 Örnek: 
- Balance değişkeni private
- Sadece fonksiyonlarla erişim sağlanır

---

### 2.  [Inheritance (Kalıtım)](src/inheritance)
Bir sınıfın başka bir sınıftan özellik ve davranış almasıdır.

 Örnek:
- Dog sınıfı Animal sınıfından türetilmiştir.

---

### 3.  [Polymorphism (Çok Biçimlilik)](src/polymorphism)
Aynı fonksiyonun farklı sınıflarda farklı davranmasıdır.

 Örnek:
- Circle ve Square farklı draw() implement eder.

---

### 4.  [Abstraction (Soyutlama)](src/abstraction)
Gereksiz detayları gizleyip sadece gerekli davranışları göstermek.

 Örnek:
- Vehicle abstract class
- Car implementation

---

##  Çalıştırma

Main.kt çalıştırılarak tüm örnekler test edilebilir.
